
# Weather App 🌤️

A simple weather app to check current and forecasted weather.

# How to start
 ```bash
1. npm install

2. npm run dev
``` 

## Features
- Displays current city weather by default
- 5-day forecast with hourly (3-hour) details
- Favorite cities: add, remove, and view weather
- Toggle between Fahrenheit & Celsius
- Uses [OpenWeatherMap API](https://openweathermap.org/) 

## Usage
1. Open the app
2. View current or choosen city weather
3. Select a day for hourly forecast
4. Add/remove cities to favorites and click to see their weather
5. Toggle temperature units as needed


## Built With

* [React](https://reactjs.org/)
